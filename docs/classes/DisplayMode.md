## Static Variables

| Field Name | Type | Initial Value | Description |
| ------------ | ------ | --------------- | ------------- |
| Borderless | Int | `1` | Windowed with borders hidden. |
| Fullscreen | Int | `2` | Full screen mode. |
| Windowed | Int | `0` | Fully windowed mode. |
