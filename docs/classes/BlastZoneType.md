## Static Variables

| Field Name | Type | Initial Value | Description |
| ------------ | ------ | --------------- | ------------- |
| ALWAYS_KO | Int | `0` | The blast zone will always KO a character |
| DISABLED | Int | `2` | The blast zone will never KO a character |
| HURT_KO | Int | `1` | The blast zone will only KO a character that is in a hurt state |
